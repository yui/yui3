Module: history

Provides browser history management functionality using a simple add/replace/get paradigm.

The history-base module uses a simple object to store state. To integrate state management with browser history and allow the back/forward buttons to navigate between states, use history-hash.

This module contains the following classes:

  • History
  • HistoryBase

Submodules:

history-base
Provides browser history management functionality using a simple add/replace/get paradigm.

The history-base module uses a simple object to store state. To integrate state management with browser history and allow the back/forward buttons to navigate between states, use history-hash.

history-hash
The history-hash module adds the History class, which provides browser history management functionality backed by window.location.hash. This allows the browser's back and forward buttons to be used to navigate between states.
