Number setInterval ( msec , request )
Sets up a polling mechanism to send requests at set intervals and forward responses to given callback.
Parameters:
msec <Number> Length of interval in milliseconds.
request <Object> An object literal with the following properties:

callback
An object literal with the following properties:
success
The function to call when the data is ready.
failure
The function to call upon a response failure condition.
argument
Arbitrary data payload that will be passed back to the success and failure handlers.
