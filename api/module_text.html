Module: text

Text utilities.

This module contains the following classes:

  • Text.AccentFold
  • Text.Data.AccentFold
  • Text.Data.WordBreak
  • Text.WordBreak

Submodules:

text-accentfold
Provides a basic Unicode accent folding implementation that converts common accented letters (like "รก") to their non-accented forms (like "a").
text-data-accentfold
An imperfect, incomplete reverse mapping of ASCII characters to case-insensitive regexes that match their most common accented forms.
text-data-wordbreak
Unicode character data used by the word breaking algorithm.
text-wordbreak
Provides utility methods for splitting strings on word breaks and determining whether a character index represents a word boundary.
