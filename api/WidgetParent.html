Class WidgetParent

Widget extension providing functionality enabling a Widget to be a parent of another Widget.

widget:itemAdded ( e )

Configuration Attributes

activeItem - Widget

Returns the Widget's currently focused descendant Widget.

defaultItemType - String

String representing the default type of the children managed by this Widget.

items - Array

Returns an array of all of the children that are direct descendants.

multiple - writeonce Boolean

Boolean indicating if multiple children can be selected at once.
Default Value: false

selection - {Array|Widget}

Returns the currently selected child Widget. If the mulitple attribte is set to true will return an array of the currently selected children. If no children are selection, will return null.
