Class DD.DDM - extends Base

Extends the dd-ddm-base Class to add support for the viewport shim to allow a draggable node to drag to be dragged over an iframe or any other node that traps mousemove events. It is also required to have Drop Targets enabled, as the viewport shim will contain the shims for the Drop Targets.

A reference to the currently active draggable object.

A reference to the active Drop Target

clickPixelThresh - {Number}

The number of pixels moved needed to start a drag operation, default 3.

clickTimeThresh - {Number}

The number of milliseconds a mousedown needs to exceed to start a drag operation, default 1000.

INTERSECT - Number

In intersect mode, a Drop is targeted by "part" of the drag node being over the Target

mode - Number

The mode that the drag operations will run in 0 for Point, 1 for Intersect, 2 for Strict

otherDrops - {Object}

An object literal of Other Drop Targets that we encountered during this interaction (in the case of overlapping Drop Targets)

POINT - Number

In point mode, a Drop is targeted by the cursor being over the Target

STRICT - Number

In strict mode, a Drop is targeted by the "entire" drag node being over the Target

tars - {Array}

All of the Targets

useHash - {Boolean}

Should we only check targets that are in the viewport on drags (for performance), default: true

validDrops - {Array}

An array of the valid Drop Targets for this interaction.
