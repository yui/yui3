Module: datatype

The DataType Utility provides type-conversion and string-formatting convenience methods for various JavaScript object types.

This module contains the following classes:

  • DataType.Date
  • DataType.Date.Locale
  • DataType.Number
  • DataType.XML

datatype-date-format
Format date submodule implements strftime formatters for javascript based on the Open Group specification defined at http://www.opengroup.org/onlinepubs/007908799/xsh/strftime.html This implementation does not include modified conversion specifiers (i.e., Ex and Ox)
